# What You Need
1. Python 3.7, flask, Tensorflow>=1.14, numpy, konlpy, ipython
2. facebook Messenger API, webhook
3. Ngrok(secure tunneling) or SSL cert over flask
4. 아직 SSL cert 미승인으로 인해 페이스북 페이지 연결 지연되고 있습니다...
